To go far, go together.

We know that staying motivated when learning a new language can be difficult. So we don't. There is an old saying - "If you want to go fast, go alone if you want to go far, go together." - We like to meetup as often as our agendas allow to practice the things we learn and help keep each other accountable.

We meet Mon / Wed / Fri @ 6pm venue's change so send us a message to be added to the mailing list.
